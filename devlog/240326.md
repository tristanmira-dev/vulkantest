# Frictions
Was slightly confused about layouts and bindings. Realised that these bindings are a type of collection or buffer of your vertex data, so for example, each
vertex consisting of the same attributes will be of binding 0, whilst layout is just specifying the the place of your vertex struct members (starting from 0)
so a more visual representation would look like


binding 0
struct vertex {
	(layout 0) glm::vec3 position;
	(layout 1) glm::vec3 color;
};

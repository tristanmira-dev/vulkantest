# What was achieved
- Rebuilt the graphics pipeline for revision with a different shader for the second object
- Managed to make a cube with a different shader, positioning it to the left instead of right, using a different pipeline
	- rendered the cube onto the screen by rebinding the pipeline inside the record command buffer function, and making another draw call
